## Top20 速记补丁

- **vue2和vue3的区别**：Vue 2 用 Object.defineProperty；Vue 3 用 Proxy，新增/删除属性与 Map/Set 支持更自然。
- **进程和线程的区别**：进程有独立地址空间、偏资源隔离；线程共享进程资源、偏 CPU 调度。
- **性能优化**：先用 Lighthouse/Web Vitals 定位，再从资源、渲染、运行时优化，给出指标前后对比。
- **http和https的区别**：HTTPS 在 HTTP 上加入 TLS，提供加密、身份认证与完整性保护。
- **事件循环机制**：同步代码结束后清空微任务，再取下一个宏任务；浏览器还要结合渲染机会。
- **vite和webpack的区别**：Webpack 开发时以打包为中心；Vite 基于原生 ESM 按需加载，生产构建再打包。
- **get和post的区别**：GET 侧重获取、通常可缓存且幂等；POST 侧重提交副作用。
- **==和===的区别**：== 允许类型转换后比较；=== 不做隐式类型转换。
- **深拷贝和浅拷贝的区别**：浅拷贝只复制第一层引用；深拷贝递归复制嵌套结构。
- **v-if和v-show的区别**：v-if 按条件创建/销毁；v-show 始终渲染只切换 display。
- **computed和watch的区别**：computed 是带缓存的同步派生值；watch 监听变化执行副作用。
- **浏览器缓存机制**：强缓存命中时不请求服务器；过期后通过 ETag/Last-Modified 协商。
- **浏览器渲染原理**：从 URL/缓存到 DOM、CSSOM、Render Tree，再经 Layout、Paint、Composite。
- **防抖和节流的区别**：防抖把连续触发合并为最后一次，节流限制时间窗口内最多执行一次。
- **闭包是什么？**：闭包让函数在外层执行结束后仍访问词法作用域。
- **ref和reactive区别**：ref 适合包装值且通过 .value 访问；reactive 代理对象。
- **vue响应式原理**：核心是依赖收集与触发更新：读取时 track，写入时 trigger。
- **tcp和udp的区别**：TCP 提供连接、可靠、有序传输；UDP 无连接、开销小、低延迟。
- **箭头函数和普通函数的区别**：箭头函数没有自己的 this、arguments、prototype，不能作为构造函数。
- **垃圾回收机制**：主流是可达性分析，不再可达的对象会被回收，现代引擎通常分代回收。
