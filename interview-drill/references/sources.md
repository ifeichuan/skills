# Reference Sources

## For agent

本目录是 `interview-drill` 的题库。出题前按优先级读；所有答案已内联在本目录文件中，不需要读取外部文件。

## 读序（每次开练）

1. **`vault-interview-log.md`** — 真实场次与可出题干（最高优先贴近）
2. **`nowcoder-hot-qa.md`** — 秋招清洗 45 张完整题卡（速记 + 要点 + 易错 + 完整答）
3. **`nowcoder-freq-index.md`** — 牛客全量 48262 条的高频题干/标签（只索引，不答）
4. **`bagu-topic-map.md`** → **`bagu-question-index.md`** — 八股专题导航与 640 题干

## 文件职责

| 文件 | 题量/规模 | 含答案？ | 何时用 |
|------|-----------|----------|--------|
| `vault-interview-log.md` | 场次 + ~60 题干 | 否 | 优先模拟真实追问链 |
| `nowcoder-hot-qa.md` | 45 | **是** | 默认 drill 抽卡 |
| `nowcoder-freq-index.md` | Top60 + 出题包 | 部分速记 | 按市场热度补题 |
