# Ant Design 适配器

## 搜索策略

**主策略：direct-url**
- URL 模板：`https://ant.design/components/{name}`
- 组件名格式：小写连字符，如 `date-picker`、`data-table`、`dropdown`
- 完整组件列表：`https://ant.design/components/overview`

**备用**
- 中文文档：`https://ant-design.antgroup.com/components/{name}-cn`

## 结果页特征

- 组件文档页包含：组件描述、多个 demo（内联可交互）、API 文档
- 每个 demo 有标题、描述和 live 预览
- demo 区域是截图目标

## 截图技巧

- 等待时间：3 秒（demo 渲染较慢）
- 视口建议：1280x800
- 页面通常很长，包含多个 demo，截取前 2-3 个最有代表性的 demo
- 可以展开 code 查看代码

## 已知问题

- npm 包名：`antd`
- GitHub 仓库：`ant-design/ant-design`
- 中英文文档路由不同
