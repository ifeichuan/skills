# Material UI 适配器

## 搜索策略

**主策略：direct-url**
- URL 模板：`https://mui.com/material-ui/react-{name}/`
- 组件名格式：小写连字符，如 `date-picker` → `react-date-picker`
- 完整组件列表：`https://mui.com/material-ui/all-components/`

**备用**
- 搜索：`https://mui.com/material-ui/?q={query}`

## 结果页特征

- 组件文档页包含：组件描述、多个 live demo（内联）、Props 文档
- 每个 demo 可以切换代码视图、打开 CodeSandbox
- 页面左侧有组件导航栏

## 截图技巧

- 等待时间：3 秒
- 视口建议：1280x800
- demo 区域有灰色边框，是截图目标
- 部分组件有 "Playground" 交互区域

## 已知问题

- npm 包名：`@mui/material`（核心）、`@mui/x-date-pickers`（日期选择器等）
- GitHub 仓库：`mui/material-ui`
- URL 中的组件名有 `react-` 前缀
