# React Bits 适配器

## 搜索策略

**主策略：google-site**（网站以浏览导航为主，无明显搜索框）
- 搜索模板：`site:reactbits.dev {query} component`
- 图片搜索：`site:reactbits.dev {query}` + `&tbm=isch`

**备用：直接浏览**
- 组件列表：`https://www.reactbits.dev/components`
- 文档页：`https://www.reactbits.dev/docs`

## 结果页特征

- 开源 React 动效组件库，110+ 创意组件
- 每个组件有 4 个变体
- 包含 live demo 和代码示例
- 还有设计工具：Background Studio、Shape Magic、Texture Lab
- 33.8K+ GitHub stars

## 截图技巧

- 等待时间：3 秒（动画组件渲染）
- 视口建议：1280x800
- 组件都带动画效果，截图时确保动画已启动
- Showcase 页面有组件的汇总展示

## 已知问题

- 100% 免费开源，但有 Pro 版（85+ 额外组件、100+ UI blocks、5 个模板）
- 偏重创意/动画组件，不是传统 UI 组件库
- 以浏览为主，没有传统搜索功能
