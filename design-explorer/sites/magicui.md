# Magic UI 适配器

## 搜索策略

**主策略：direct-url**（组件有明确的 URL 路径）
- URL 模板：`https://magicui.design/docs/components/{name}`
- 组件名格式：小写连字符，如 `animated-beam`、`border-beam`、`magic-card`
- 完整组件列表：侧边栏导航，或 `https://magicui.design/docs`

**备用：站内搜索**
- `Cmd+K` 唤起搜索面板
- 用 agent-browser：`agent-browser press Meta+k` 然后输入查询词

**备用：google-site**
- `site:magicui.design {query} component`

## 结果页特征

- 组件文档页包含：live 交互 demo、代码示例、安装说明
- 组件分类：Marquee、Terminal、Hero Video Dialog、Bento Grid、Dock、Globe、
  Tweet Card、Orbiting Circles、Icon Cloud、Lens、Pointer、Progressive Blur 等
- 有 Templates 页面展示完整项目模板

## 截图技巧

- 等待时间：3 秒（动画组件渲染）
- 视口建议：1280x800
- 组件 demo 区域是截图目标，通常在页面顶部
- 动画类组件（beam、orbiting circles）需要等动画启动后截图
