# Best Designs on X 适配器

## 搜索策略

**主策略：google-site**（网站本身以 feed/gallery 浏览为主）
- 搜索模板：`site:bestdesignsonx.com {query} design`
- 图片搜索：`site:bestdesignsonx.com {query}` + `&tbm=isch`

**备用：站内筛选**
- 首页有 Filter 按钮和视图切换（Feed View / Grid View / Masonry View）
- URL：`https://bestdesignsonx.com/`，使用页面上的筛选功能

## 结果页特征

- 聚合 X/Twitter 上设计师分享的设计作品，约 7500+ 作品
- 展示格式：设计截图 + 设计师 @username + 点赞/收藏
- 支持多种视图：Feed（时间线）、Grid（网格）、Masonry（瀑布流）
- 属于 Panda Network 设计策展生态

## 截图技巧

- 等待时间：3 秒（图片加载）
- 视口建议：1440x900
- Masonry 视图最适合截图（信息密度高）
- 需要向下滚动获取更多内容（分页，共 64 页）
- 截图单个设计卡片获得高清参考

## 已知问题

- 内容来源是 X/Twitter 设计师的分享，风格多样
- 没有精确的组件级搜索，主要靠 Google `site:` 匹配
- 关联网站：CollectUI、DDR、Curations Supply（同属 Panda Network）
- 可能有赞助商/广告内容混在设计作品中
